Option A: Scalar (The Modern 2026 Favorite)
Scalar is often preferred over Swagger now because it is faster and has a cleaner design.
Install: dotnet add package Scalar.AspNetCore
Code:
csharp
app.MapOpenApi(); // The Data
app.MapScalarApiReference(); // The UI
Use code with caution.

Access: http://localhost:5122/scalar/v1


Option B: Swagger UI (The Classic)
Install: dotnet add package Swashbuckle.AspNetCore.SwaggerUi
Code:
csharp
app.MapOpenApi();
app.UseSwaggerUI(options => {
    options.SwaggerEndpoint("/openapi/v1.json", "v1");
});
Use code with caution.

Access: http://localhost:5122/swagger

Summary for your Journey:
Beginner: Leaves it open to everyone.
Intermediate: Uses if (app.Environment.IsDevelopment()).
Master: Uses ASP.NET Core Identity or a Policy-based approach with an API Key stored in appsettings.json or Azure Key Vault.(the ui will hv an authorize btn to input Bearer token)

If you just want a quick password for a private project, I recommend the Basic Auth Middleware approach, which triggers the standard browser username/password(i.e before u even see the api)

app.MapWhen((context) => 
{
    // This is a Func<HttpContext, bool>, so we MUST return a boolean.
    return context.Request.Path == "/" ||                          
           context.Request.Path.StartsWithSegments("/scalar") ||   
           context.Request.Path.StartsWithSegments("/openapi");
}, (branch) => 
{
    // This is an Action<IApplicationBuilder>. 
    // It is 'void', so we DO NOT use a return statement here.

    branch.Use(async (context, next) => 
    {
        string authHeader = context.Request.Headers["Authorization"];

        if (authHeader != null && authHeader.StartsWith("Basic ")) 
        {
            var encodedHeader = authHeader["Basic ".Length..].Trim();
            var credentials = System.Text.Encoding.UTF8
                .GetString(Convert.FromBase64String(encodedHeader))
                .Split(':');

            if (credentials.Length == 2)
            {
                var username = credentials[0];
                var password = credentials[1];

                if (username == "admin" && password == "123456") 
                {
                    await next(); 
                    return; // This exits the middleware early because the job is done.
                }
            }
        }

        context.Response.Headers["WWW-Authenticate"] = "Basic realm=\"Secure API Documentation\"";
        context.Response.StatusCode = 401; 
        
        // No return needed here because it's the end of a 'Task' (void) function.
    });

    branch.UseRouting();
    branch.UseEndpoints(endpoints => 
    {
        // These are Actions, so no return statements allowed inside these blocks.
        endpoints.MapGet("/", () => "Welcome to the Protected API Root!");
        endpoints.MapOpenApi();
        endpoints.MapScalarApiReference();
    });

});
